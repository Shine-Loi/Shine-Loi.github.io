# 1. Bitwise Operators
You can perform bit-level operations by using bitwise operators. Python will return 0 if the result is false or 1 if true.

- &: Bitwise AND.
- \|: Bitwise OR.
- ^: Bitwise XOR.
- ~: Bitwise NOT.
- \<\<: Bitwise left shift.
- \>\>: Bitwise right shift.

&nbsp;

```python
a=25
b=13

print("a: ", bin(a))
print("b: ", bin(b))
print("a&b: ", a&b)
print("a|b: ", a|b)
print("a^b: ", a^b)
print("~a: ", ~a)
print("a<<2: ", a<<2)
print("a>>2: ", a>>2)
```

&nbsp;

| Output |
|---|
| a:  0b11001 |
| b:  0b1101 |
| a&b:  9 |
| a\|b:  29 |
| a^b:  20 |
| ~a:  -26 |
| a\<\<2:  100 |
| a\>\>2:  6 |
